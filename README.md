# EngineX

AI-powered SEO SaaS platform — self-hosted on African Soil.

## Stack
- Traefik — Gateway + SSL
- PostgreSQL + Redis — Database
- seo-tools-api — SEO Core
- SerpBear — Daily Rank Tracking
- OpenPanel — Analytics
- Grafana — Unified Dashboard
- NestJS — Auth + Multi-tenancy
- Uptime Kuma — Monitoring
- docker-volume-backup — Backups

## Quick Start
git clone --recurse-submodules https://github.com/boka-o1/enginex
cd enginex
make up
